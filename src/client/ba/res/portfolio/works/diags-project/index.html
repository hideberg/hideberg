Diags Project

Deliver your house inspection faster and easy

  • web app / platform

Description

Diags Project is a freelance project that connect people who want to order a house inspection and inspectors to work through the web app. It works out of the box and consists in a intuitive booking system with realtime inspector availability and a back-office to track orders status and invoicing. The clients can be house owners, real state agencies or other companies.

How

Diags consist in one application and his separated backend. In the web application, a person (landlord or a company) can order a house inspection with an easy to use booking form. After this first step, he becomes an user with access to a back-office when he can se his order/s status and monthly invoice (companies only). At the same type, any inspector interested in the use of this platform for work can register as an Inspector after negotiating a revenue porcentage with the web app admin.
